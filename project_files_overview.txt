MoneyPal Project - File Overview
================================

CORE APPLICATION FILES:
-----------------------
app.py - Main Flask application with all routes and functionality
  - Dashboard route with financial summaries and recent transactions
  - Cash flow CRUD operations (create, read, update, delete)
  - CSV export functionality
  - Database configuration using Documents folder path

models.py - SQLAlchemy database models
  - CashFlow model with fields: id, date, amount, description, category, notes
  - Database table structure definition



TEMPLATES (HTML FILES):
----------------------
templates/dashboard.html - Main dashboard page
  - Financial summary display
  - Recent transactions list
  - Category breakdown charts
  - Navigation to other pages

templates/cashflow.html - Transaction management page
  - Add new transactions form
  - Edit/delete existing transactions
  - Transaction history table
  - Filtering and sorting options



CONFIGURATION FILES:
-------------------
requirements.txt - Python package dependencies
  - Flask==3.0.0
  - Flask-SQLAlchemy==3.1.1
  - python-dotenv==1.0.0

.env.example - Environment variables template
  - Sample configuration for Flask settings
  - Database URL configuration
  - Secret key placeholder

.gitignore - Git version control exclusions
  - Python cache files
  - Virtual environment folders
  - IDE configuration files
  - OS-specific files
  - Database files (commented out to allow tracking)

UTILITY FILES:
-------------
seed_sample_data.py - Database seeding script
  - Generates sample transaction data for testing
  - Creates realistic financial entries with categories
  - Configurable date ranges and amounts
  - Uses Documents folder database path

check_requirements.py - Dependency verification script
  - Checks if all required Python packages are installed
  - Validates Python version compatibility
  - Reports missing dependencies

run_moneypal.bat - Windows batch file launcher
  - One-click application startup
  - Installs dependencies if needed
  - Launches Flask development server

DOCUMENTATION FILES:
-------------------
README.md - Project documentation
  - Feature overview and description
  - Installation and setup instructions
  - Usage guidelines
  - Technology stack information
  - Project roadmap and status

CHANGELOG.md - Version history
  - Record of changes and updates
  - Feature additions and bug fixes
  - Version numbering and dates

ROADMAP.md - Future development plans
  - Planned features and enhancements
  - Development phases and milestones
  - Priority levels for new functionality

tips.md - Usage tips and best practices
  - Application usage guidelines
  - Tips for effective financial tracking
  - Troubleshooting common issues

CACHE/TEMPORARY FILES:
---------------------
.mypy_cache/ - MyPy type checker cache directory
  - Contains type checking metadata
  - Automatically generated files
  - Can be safely deleted and regenerated

DATABASE LOCATION:
-----------------
The application stores data in:
C:\Users\[username]\Documents\MoneyPal\cashflow.db

This SQLite database contains:
- cash_flow table with all financial transactions
- Automatically created on first run
- Shared between all application instances

PROJECT STRUCTURE SUMMARY:
--------------------------
- Web-based Flask application for personal finance tracking
- SQLite database for data persistence
- HTML templates for user interface

- Sample data generation for testing
- Comprehensive documentation and setup files
- Windows-optimized with batch file launchers
